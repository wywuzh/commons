# commons - 项目公共组件类库
本项目公共组件类库主要依赖`spring-boot`，**为了便于使用对应版本的spring-boot，本版本将会使用和spring-boot相同的版本号。**


### 依赖资源
| 依赖资源 | 依赖版本 |
| ------- | ------- |
| spring-boot | 2.1.5.RELEASE |
| spring | 5.1.7.RELEASE |
| aspectj | 1.9.2 |
| junit | 4.12 |
| mockito | 2.23.4 |
| mybatis | 3.5.2 |
| mybatis-spring | 2.0.1 |
| mybatis-spring-boot | 2.0.1 |
