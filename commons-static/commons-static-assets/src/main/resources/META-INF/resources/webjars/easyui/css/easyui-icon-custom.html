jQuery-EasyUI自定义过图标

图标使用必须先调用easyui-icon样式，例如：
  <span class="easyui-icon icon-new"></span>
或者
  <i class="easyui-icon icon-new"></i>

添加自定义图标，添加如下样式：
.icon-date{
    background:url('../image/icon/date.gif') no-repeat center center;
}
